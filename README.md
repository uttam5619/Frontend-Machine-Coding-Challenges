# React Machine Coding Workspace

This repository contains multiple React practice projects organized by question type.

## Folder Structure

- `fundamental_questions/` - foundational React exercises
- `standard_questions/` - commonly asked machine coding problems

## Getting Started

1. Open a project folder.
2. Install dependencies:
   - `npm install`
3. Run the project:
   - `npm run dev`

## Notes

- `node_modules` and other generated files are ignored through the root `.gitignore`
